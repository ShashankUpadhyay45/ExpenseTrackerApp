# SpendSage

A clean offline-first expense tracker built using HTML, CSS and JavaScript.

## Features

- Add expenses
- Recurring expenses
- Search & filtering
- Monthly analytics
- Category breakdown
- Yearly breakdown
- CSV export
- Dark mode
- Local storage persistence

## Technologies

- HTML5
- CSS3
- JavaScript
- LocalStorage
- Canvas API

## Future Improvements

- CSV Import
- IndexedDB
- Budget Tracking
- Income Management
- Cloud Sync
- Authentication
